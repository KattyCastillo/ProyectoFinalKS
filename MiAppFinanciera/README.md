## 📋 Resumen Ejecutivo

**Mi App Financiera** es una aplicación React Native con TypeScript que implementa un gestor de finanzas personales enfocado en:

- 💳 **Gestión de Tarjetas de Crédito**
- 👫 **Finanzas Compartidas** (Pareja)
- 💰 **Seguimiento de Gastos**
- 🎁 **Optimización de Beneficios** (Puntos, Millas, Cashback)

## 🚀 Inicio Rápido

### 1. Instalación
```bash
cd MiAppFinanciera
npm install
```

### 2. Ejecutar el Proyecto
```bash
npm start           # Terminal 1: Metro bundler
npm run ios         # Terminal 2: Simulador iOS
# o
npm run android     # Emulador Android
```

### 3. Credenciales de Prueba
- **Email**: cualquier email válido (ej: test@example.com)
- **Contraseña**: cualquier contraseña con 6+ caracteres
